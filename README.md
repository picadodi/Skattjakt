# Skattjakt
Ett python spel där spelaren ska försöka hitta så många gömda skatter som möjligt och undvika fällorna, men akta så du inte tar slut på dina steg!

Skapa din egna personliga karta genom att köra spel.py filen och sedan andvända WASD-kommando för att röra dig runt :)

**HINT: En fälla och en skatt kan inte placeras bredvid varandra varken diagonalt, horisontellt eller vertikalt.
